Portfolio

Factorial Finder

My First Website

Back in 9th grade, I created my very first website. At the time, I had only a little exposure to HTML,
CSS, and JavaScript through YouTube tutorials, but I was curious to build something useful.
Since I loved mathematics, I decided to work on a problem I often faced—calculating factorials.

Manually solving factorials used to take me a lot of time, so I built a simple website that
could calculate the factorial of any number instantly. This project not only made my math practice
easier but also gave me the confidence to continue learning web development
and problem-solving through coding.

E-Commerce Website

Full-Stack E-Commerce Website

This project is a fully functional e-commerce website that delivers the core experience of a modern online store. I created this website during my 12th grade for my father’s company, Alam Enterprises, a furniture manufacturing business. Although it was never used commercially and left for testing purposes, it took a lot of time and effort to complete. Visitors can browse through a variety of products, view them in a clean gallery layout, and quickly find what they need using the built-in search bar. The shopping cart makes the process smooth and interactive, allowing users to add, remove, and review items with ease before checkout.

Beyond shopping, the website also features a dedicated blogs section for sharing updates, ideas, and product-related insights, adding extra value for users. To make communication effortless, it includes multiple contact options such as WhatsApp, Instagram, and Facebook, each equipped with pre-written messages for instant connection. Users can easily log in to the website, buy products, track their orders, and even pay bills securely. While still in its testing phase, the site is already live and demonstrates nearly all the features expected from a complete e-commerce platform.
